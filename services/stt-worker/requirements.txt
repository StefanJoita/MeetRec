#   pip install torch             → descarcă versiunea CUDA (~2.5GB!)
#   pip install torch --index-url https://download.pytorch.org/whl/cpu
#                                 → descarcă versiunea CPU-only (~800MB)
#
# Versiunea CPU-only se specifică în Dockerfile cu --extra-index-url.
# Dacă rulezi pip local (fără Docker) și nu ai GPU, adaugă manual:
#   pip install torch==2.1.0+cpu --index-url https://download.pytorch.org/whl/cpu
# ============================================================
